### What it is

Swiss One Lab designs **coherent, secure, automation-first platform environments** for professional operations.

- Secure, multi-environment platform architecture
- Automation, observability &amp; governance built in by design
- AI-assisted operations — ship faster, more reliably
- **Local-first &amp; sovereign** — software that keeps data under its owner's control

---

### What we build

| Domain | Focus |
|---|---|
| **Platform Engineering** | Modular multi-environment infrastructure, IaC, controlled delivery |
| **Security Architecture** | Hardening, threat modeling, governance &amp; compliance |
| **DevSecOps Automation** | CI/CD pipelines, observability, automated maintenance |
| **Cybersecurity Operations** | Pentest, OSINT, forensics, reverse engineering |
| **Sovereign &amp; Local-First Apps** | Offline-capable, privacy-first software; data ownership by design |
| **AI Agents &amp; Local LLMs** | On-device LLM orchestration (Ollama, MLX), agentic engineering workflows |

### Principles

- **Security-first** — hardening and governance built in, not bolted on
- **Sovereign by design** — local-first, offline-capable; you own your data
- **Modular** — composable tools over monolithic lock-in
- **AI-assisted** — local models and agents augment the craft
